# quickSort
Contains a QT project for a Windows standalone app for viasualization of the Quick Sort Algorithm 

## Execute standalone version on Windows

- Download the repository 
- Extract all files
- Open the directory `..\quickSortAlgorithm\build\Desktop_Qt_6_10_1_MinGW_64_bit-MinSizeRel`
- Excecute `appquickSortAlgorithm`
